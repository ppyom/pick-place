# Icon

`lucide-react` 아이콘을 프로젝트 전용 이름(`IconName`)으로 감싸서 사용하는 컴포넌트입니다.
디자인 시스템의 `IconSize` 토큰(`sm` / `md` / `lg`)에 맞춰 크기가 고정됩니다.

## Stroke Width

기본값은 `2`입니다. 강조가 필요한 곳에서만 조정하고 웬만하면 기본값을 그대로 사용하는 걸 권장합니다.

<Canvas of={IconStories.StrokeWidth} />

## Icon이 없을 때

`name`에 해당하는 아이콘이 `ICONS`에 없거나 값이 `null`(예: `empty`)이면 같은 크기의 빈 영역만 렌더됩니다.
아이콘이 있을 때와 없을 때 레이아웃이 흔들리지 않도록 하기 위한 처리입니다.
