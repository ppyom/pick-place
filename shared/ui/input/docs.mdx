# Input

텍스트 입력을 받는 폼 컨트롤입니다. `Default` / `Error` / `Disabled` 상태를 지원하며, 왼쪽에 아이콘을 붙일 수 있습니다. `forwardRef`를 지원해 React Hook Form 등 폼 라이브러리와 연동할 수 있습니다.

## Left Icon

`leftIcon` prop에 `IconName`을 넘기면 왼쪽에 아이콘이 표시됩니다. 아이콘 컬러는 status에 따라 자동으로 바뀝니다(`default` → brand, `error` → error).

## Usage

### 기본 사용

```tsx
<Input placeholder="장소를 검색해보세요" />
```

### 아이콘과 함께

```tsx
<Input leftIcon="search" placeholder="장소를 검색해보세요" />
```

### 에러 상태

```tsx
<Input status="error" defaultValue="이미 사용 중인 이름이에요" />
```

### React Hook Form 연동

`forwardRef`를 지원하므로 `register`를 그대로 스프레드할 수 있습니다.

```tsx
const { register } = useForm();

<Input {...register('name')} placeholder="이름" />;
```
