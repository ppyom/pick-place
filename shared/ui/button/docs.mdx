# Button

`primary` / `secondary` / `outline` / `text` 4가지 variant와 `sm` / `md` / `lg` / `icon` 4가지 크기를 지원하는 버튼 컴포넌트입니다.
아이콘은 별도 prop이 아니라 `children`으로 텍스트와 함께 직접 조합해서 사용합니다.

## With Icon

아이콘은 `children`에 `Icon` 컴포넌트를 텍스트와 함께 직접 배치합니다.

```tsx
<Button>
  <Icon name="save" size="sm" />
  저장하기
</Button>
```
