# Avatar

프로필 이미지를 원형으로 표시하는 컴포넌트입니다. 이미지 로드에 실패하면 이니셜 또는 아이콘으로 대체(fallback)됩니다.

## Fallback 우선순위

- `src` 로드 성공 시 이미지를 표시합니다.
- 로드 실패하거나 `src`가 없으면 `fallbackText`를 사이즈별 글자 수만큼 잘라 이니셜로 표시합니다.
- `fallbackText`도 없으면 기본 아이콘을 표시합니다.
