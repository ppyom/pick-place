# Chip

`filled` / `outline` / `soft` 3가지 외형과 `sm` / `md` 2가지 크기를 지원하는 Chip 컴포넌트입니다.
아이콘은 별도 prop이 아니라 `children`에 텍스트와 함께 직접 배치해서 사용합니다.

## With Icon

아이콘은 `children`에 `Icon` 컴포넌트를 텍스트와 함께 직접 배치합니다.

```tsx
<Chip selected>
  <Icon name="save" size="sm" />
  저장
</Chip>
```
