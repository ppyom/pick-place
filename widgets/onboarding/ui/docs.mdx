# Onboarding

관심사 → 동행인 → 관심 지역, 3단계로 선호도를 수집하는 온보딩 위젯입니다. 스텝 정의(config)를 데이터로 두고 컨테이너가 순회하며 렌더하는 **config-driven** 구조입니다.
