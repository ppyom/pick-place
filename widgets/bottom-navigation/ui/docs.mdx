## Safe Area

하단 여백에 `pb-[env(safe-area-inset-bottom)]`를 적용해 iOS 홈 인디케이터와 겹치지 않도록 처리했습니다.
데스크톱 브라우저에서는 값이 항상 0으로 나와 시각적으로 확인이 불가능하며 실제 검증은 iOS 시뮬레이터 또는 실기기에서 진행합니다.
